## 🏃 How to execute

Create the .env file with the same variables as the .env.example file.

Clone project:

```bash
$ git clone https://github.com/cassiosilva93/finance-api.git
```

Install dependencies:

```bash
$ npm install
```

Execute migrations

```bash
$ npm run prisma:migrate
```

Start server:

```bash
$ npm run start:dev
```

<br>
